This is an web unit test for PRADO web components. 

It runs on any modern web browser.

To run it, unzip selenium.zip to the current directory under a selenium directory, e.g.

This directory should contain

js\          <-- compressed javascript files
selenium\    <-- ## put unzipped files here ##
web\         <-- prado web pages for testing
web_tests\   <-- client-side test scripts
config.php   <-- simpletest unit tester configs
index.php    <-- prado web page entry point
readme.txt   <-- this file
selenium.zip <-- selenium web functional test files

To run test web functional tests, point your browser to

http://yourhost/prado/tests/web_tests/

Click on "Selected Test" or "All Tests" to start web functional testing.
